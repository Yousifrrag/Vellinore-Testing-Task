Part A — Odoo Backend Product Fields (Vellinore Tab)

The "Vellinore" tab in Odoo backend contains custom fields that control the premium website layout. Fill them in backend and verify their website display:

1. Basic Product Info Group

Field NameDescription / ExampleWebsite Display & Behavior
Fabric / Material Description Composition (e.g. "100% European linen") Shows in the Quick View popup under the product title.
Weight / Thread Strength Fabric weight descriptor (e.g. "320g/m²") Stored in system for filtering and search.
Origin Material source (e.g. "Scotland") Stored in system for filtering and search.
Exclusivity Badge Tag label (e.g. "Limited" or "Inner Circle") Shows as a gold badge on the top-right corner of product grid cards and inside Quick View. Hides if empty.
Fit Scale (1–5) Numeric scale for fit tight-to-loose range. Stored in system for filtering.
Fit Details Sizing text (e.g. "Our model is 186 cm and wears size M.") Stored in system.
Fit Type Selection: Slim Fit, Regular Fit, or Oversized. Shows a visual fit bar on the product detail page:
  • Slim: dot at left + "Fällt schmal aus"
  • Regular: dot at middle + "Fällt normal aus"
  • Oversized: dot at right + "Weit geschnitten"
Hides completely if left unselected.

Part B — Website Page Testing Checklist

1. Homepage (/) Must Test

Hero video: Video plays automatically in loop (no sound). Title "Die Eleganz der Stille" and CTAs load correctly.
Season cover: Background image, "Stille am Wasser." text, and collection/lookbook links load.
Stück der Saison: Displays the designated seasonal product dynamically from the database with current pricing.
Category grid: 6 categories show with campaign photos. Links redirect correctly.
Shoppable Look: lifestyle photo with active dots. Hovering/clicking dots displays correct product tooltips with price.
Material Macro & Craft: Editorial layout images and text load with scroll-in animations.
Lookbook filmstrip: Horizontal drag-scroll track loads 5 photos with labels.
Featured carousel: swipeable product block loads live storefront products.
Stimmen (Testimonials): 3 customer quotes appear with source details.
Trust strip: 4 shipping/payment/return policy indicators display.

2. Shop Listing (/shop) Must Test

Shop banner: Header hero displays background photo and "Alle Stücke" title.
Toolbar: "Filtern" sidebar triggers, and search bar filters products.
Product cards: 3-column clean cards display image (3:4 ratio), category, name, and price with tax note.
Hover swap: Card image switches to alternate photo on hover.
Quick view trigger: "Schnellansicht" button appears on hover and opens modal.
Badges: Exclusivity label shows on top-right of image, and price reduction shows original price crossed out.

3. Product Detail Page Must Test

Media gallery: Thumbnail strip on left scrolls/changes main image on right.
Attributes: Color swatches and sizes work, updating price and image dynamically.
Size Guide: "Größenberater" link opens popup. Choosing a garment type and entering dimensions calculates size. Clicking apply closes popup.
Fit bar & specs: Visual slider dot and specs row match backend Odoo settings.
CTA buttons: "In den Warenkorb" slides cart open. "Direkt-Kauf" redirects to checkout.
Out of stock: "Benachrichtigen" form shows when a variant is out of stock. Submitting details registers request in backend.
Accordions: 3 expandable/collapsible details section toggle open/close.

4. Quick View Popup Must Test

Launch: Clicking quick view on shop card opens overlay correctly.
Interaction: Swatches change images, sizes activate buy buttons, and out-of-stock displays notify field.
Wishlist: Heart button adds/removes item to/from user selection.

5. Cart (Drawer & Page) Must Test

Cart drawer: Slides open from right, showing items, totals, checkout button.
Cart page (/shop/cart): shows item rows with quantity adjusters (+/−), remove button, and right summary column.
Monogram line: Monogram custom choices print under product name. Separates identical products with different monograms.

6. Checkout & Confirmation Must Test

Success page: Displays checkmark, order number, thank you heading, and back-to-shop link.

7. Header & Navigation Must Test

Mega menu: Hovering parent menu links loads full-width column layout with campaign images.
Konto: Dropdown shows login/register options when logged out, and dashboard/logout options when logged in.
Mobile Drawer: Left drawer slides in on mobile. Hamburger links, sub-menus (+/−), and language switcher load correctly.

8. Footer Should Test

Sections: Circular newsletter sign up, service badges, 4 columns of page links, payment marks, and copyright load.

9. Portal Dashboard (/my) Should Test

Tabs: Übersicht (overview), Bestellungen (orders), Die Auswahl (wishlist), Adressen (addresses), and Einstellungen (profile settings) load and edit correctly.

Part C — Odoo Backend Systems

Size Mapping: Go to Website → Configuration → Size Mappings. Seed tables should display measurement thresholds.
Stock Notifications: Go to Website → eCommerce → Stock Notifications. Registered out-of-stock emails should save here.
Monogram orders: Confirm monogram selections appear as custom columns on Sale Order lines.
Restock trigger: restock a variant (stock move quantity > 0). Verify that an email gets sent and the record changes to "Notified".

Part D — Responsive, Browser & Language

Mobile check: all grids wrap, columns collapse, and swipe actions work.
Language check: Switching site to EN or DE translates all system labels.
General check: No red console warnings, no broken links, and fonts load correctly.
